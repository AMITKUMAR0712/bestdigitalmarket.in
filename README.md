# Digital Future Marketing Agency Website

A premium Next.js App Router website for lead generation across SEO, paid ads, social media, website development and performance marketing services.

## Tech Stack

- Next.js 15 App Router
- React
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber
- React Icons
- Nodemailer

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Email Setup

Create `.env.local` from `.env.local.example` and add Gmail SMTP values:

```bash
GMAIL_USER=your_gmail_address@gmail.com
GMAIL_APP_PASSWORD=your_gmail_app_password_without_spaces
CONTACT_RECEIVER_EMAIL=lead_receiver_email@gmail.com
```

The contact form sends enquiries through `/api/contact` without saving leads to a database.
